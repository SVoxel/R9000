Fast Lane

The first time you turn on Video Mode, you should specify the bandwidth you want to reserve and set up the push button for Turbo Video. In other words, fill in all the fields on this screen and click Apply so that your changes take effect.

This field shows if Turbo Video is currently turned on or off.

Turn on Turbo Video:
By default, Turbo Video is turned off and the push button on front panel works as a WPS (Wi-Fi Protected Setup) button. To turn on Turbo Video, select this check box.

Bandwidth Reserved for Trusted IP Address
Specify what percentage of the bandwidth on the wireless router to reserve when you connect from a trusted IP address. This prevents others from using up your bandwidth. When you connect to the router from the trusted IP address, you have guaranteed bandwidth for video steaming applications. There are 4 options: 50%, 60%, 70%, and 80%.

Push Button:
You can set up the push button on front panel to work two ways: (1) WPS (default); (2) Turbo Video. Select the radio button for the option that you want.
